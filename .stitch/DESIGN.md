# Prudentia — Heritage Modern Design System

Generated by Stitch (project `18219485933790099597`), screen "home1".

## 1. Design Philosophy
Warm, premium, airy — light-mode corporate with generous whitespace and elegant serif typography. Inspired by the Aurika Wellness aesthetic applied to Prudentia's brand palette.

## 2. Color Palette
| Token | Hex | Usage |
|-------|-----|-------|
| Navy | `#002747` | Stats strip, CTA bg, footer, headings |
| Blue | `#00558F` | Primary links, buttons, italic accents |
| Green | `#068140` | Accent only — tags, icons, card borders, checkmarks |
| Cream | `#F8F7F3` | Page background, alternating sections |
| White | `#FFFFFF` | Cards, nav on scroll |

## 3. Typography
- **Headings** — Playfair Display (serif), weights 400/600/700, italic for decorative spans
- **Body / UI** — Inter (sans-serif), weights 400/500/600/700
- H1: 64px / 1.08 line-height
- H2: 42px / 1.2 line-height
- Section tag: 11px, bold, uppercase, 0.16em letter-spacing

## 4. Spacing & Layout
- Max container: 1280px, `mx-auto px-6`
- Section vertical padding: `py-24`
- Card border-radius: `rounded-2xl` (16px)
- Nav height: 72px (fixed)

## 5. Component Patterns
- **Section tag**: green line + uppercase label, 11px Inter bold
- **Service cards**: white bg, `border-t-4 border-[#068140]`, green dot feature list
- **Stat counters**: count-up animation on scroll-into-view
- **Bento grid**: 4-col × 2-row, first item spans 2×2, hover reveals label
- **Testimonials**: AnimatePresence crossfade, auto-advance 5s, dot navigation
- **CTA section**: navy + radial green glow (`opacity-[0.12] blur-[100px]`)
- **Gradient button**: `linear-gradient(135deg, #068140, #00558F)`

## 6. Design System Notes for Stitch Generation
When generating new screens for this project, use:
- Background: `#F8F7F3` (warm cream, not pure white)
- Headings: Playfair Display serif, navy `#002747`
- Accent: `#068140` green for tags, borders, icons only — not fills
- Primary action: gradient green→blue `linear-gradient(135deg, #068140, #00558F)`
- Cards: white bg, subtle shadow, 16px radius, green top-border on feature cards
- Stats/CTA/Footer: solid navy `#002747`
